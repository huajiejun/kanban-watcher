# 菜单栏摘要增强-测试概述

## 测试日期

2026-03-24

## 测试目标

验证菜单栏摘要增强的三条链路：

- 远端 `FetchAll` 能生成 `MenuSummary`
- 本地 `/api/workspaces/active` 能复用摘要并输出 `menu_summary`
- 托盘菜单能按“标题 + 单行摘要”格式渲染，并对长摘要截断

## 执行命令

```bash
go test ./internal/api ./internal/tray ./internal/store ./internal/sync ./internal/wechat
go test ./...
```

## 测试结果

- `internal/api` 通过
- `internal/tray` 通过
- `internal/store` 通过
- `internal/sync` 通过
- `internal/wechat` 通过
- `go test ./...` 全量通过

## 关键验证点

- `internal/api/menu_summary_test.go`
  - 验证原因提示优先于最近消息
  - 验证 markdown/code block 清洗
  - 验证无可读文本时返回空摘要
- `internal/api/client_test.go`
  - 验证 `FetchAll` 会填充 `MenuSummary` / `MenuSummaryBy`
- `internal/api/messages_test.go`
  - 验证本地摘要构建可回退到最近消息
  - 验证 `menu_summary` 会进入活跃工作区响应 JSON
- `internal/store/store_test.go`
  - 验证活跃工作区查询会带出 `last_message`
- `internal/tray/workspace_menu_formatter_test.go`
  - 验证摘要行显示
  - 验证长文本截断
  - 验证空摘要时回退为单行
