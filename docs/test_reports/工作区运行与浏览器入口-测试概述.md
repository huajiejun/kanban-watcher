# 工作区运行与浏览器入口-测试概述

- 2026-03-26 10:24:13 +0800

## 验证范围

- 工作区卡片运行按钮
- 工作区主页“打开浏览器”按钮
- `POST /api/workspace/{id}/dev-server` 后端接口
- 活跃工作区摘要 `browser_url` 返回

## 结论

- 工作区卡片已支持“运行”按钮，且不会影响主选择按钮行为
- 工作区主页已支持按当前激活工作区打开 `browser_url`
- 后端已支持代理启动 dev server，并在活跃工作区摘要中返回 `browser_url`

## 已知约束

- `browser_url` 由 `config.yaml` 中 `http_api.browser_url_template` 生成；若模板为空，“打开浏览器”按钮会保持禁用
