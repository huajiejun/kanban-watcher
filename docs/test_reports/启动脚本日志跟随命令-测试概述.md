# 启动脚本日志跟随命令-测试概述

## 测试目标

验证 `scripts/start-dev.sh logs <worktree_id>` 会持续跟随后端与前端日志文件。

## 验证命令

```bash
bash scripts/test_start_dev_logs_command.sh
```

## 结果

通过

输出：

```text
PASS: logs 命令会持续跟随后端和前端日志
```

## 覆盖点

- `logs` 子命令会调用 `tail -F`
- 会同时跟随后端日志 `/tmp/kanban-backend-<port>.log`
- 会同时跟随前端日志 `/tmp/kanban-frontend-<port>.log`
