# 网页版工作区实现计划

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增网页版工作区主入口，首页展示左侧项目状态栏和右侧最多 4 个同步刷新的工作区窗格，同时将原预览页迁移到 `/preview` 路由并复用现有卡片/会话内容能力。

**Architecture:** 保留现有 `kanban-watcher-card` 作为 Home Assistant 卡片容器；新增网页入口壳层负责首页和 `/preview` 路由；把工作区分组列表和会话内容从卡片容器中抽为可复用单元，再由网页壳层编排“自动打开 / 手动关闭 / 第 5 个顶替最早窗格 / 多窗格同步刷新”等状态。

**Tech Stack:** Vite、TypeScript、Lit、Vitest、现有 HTTP API / WebSocket 实时能力

---

export type PageMode = "workspace" | "preview";

export function getPageMode(url = new URL(window.location.href)): PageMode {
  return url.pathname === "/preview" || url.pathname === "/preview/"
    ? "preview"
    : "workspace";
}
