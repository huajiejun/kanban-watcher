# 手机模式 Realtime 地址修复计划

**Goal:** 让浏览器手机模式下的内嵌 `kanban-watcher-card` 与桌面模式一致，优先使用 `/api/info` 返回的 `realtime.base_url` 建立 websocket，避免错误连到当前 worktree 后端。

**Architecture:** 在卡片配置与运行时状态中补充 `realtime_base_url`，初始化时读取 `/api/info` 的 realtime 地址，并在板级与会话级 websocket 建连时优先使用该地址。保持现有 HTTP 请求继续走 `base_url`，仅分流 realtime 通道。

**Tech Stack:** TypeScript、Lit、Vitest
