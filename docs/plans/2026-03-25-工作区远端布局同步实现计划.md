# 工作区远端布局同步实现计划

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为网页版工作区增加远端布局持久化，并让需要注意任务按 attention 轮次自动弹出与抑制。

**Architecture:** 后端新增工作区布局读写接口和数据库表，前端在本地缓存之上增加远端读写同步层；attention 自动弹出逻辑继续集中在 `workspace-page-state`，但把 dismiss 语义从简单 ID 集合升级为“按轮次抑制”。

**Tech Stack:** Go、MySQL、Lit、TypeScript、Vitest
