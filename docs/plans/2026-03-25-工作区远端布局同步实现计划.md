# 工作区远端布局同步实现计划

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为网页版工作区增加共享布局的远端持久化和 websocket 实时广播，并让需要注意任务在多端间保持一致的自动弹出与 dismiss 语义。

**Architecture:** 后端新增共享布局表、读写接口和 realtime 广播事件，前端在本地缓存之上增加远端布局 hydrate、HTTP 写回和 websocket 收敛层；attention 自动弹出逻辑继续集中在 `workspace-page-state`，dismiss 仍使用 ID 集合，但其生命周期由 attention 跃迁与手动重新打开共同维护。

**Tech Stack:** Go、MySQL、Lit、TypeScript、Vitest
