# 2026-03-24 mac应用打包

## 目标

为 `kanban-watcher` 增加可重复执行的 macOS `.app` 打包能力，产出可双击启动的 `Kanban Watcher.app`。

## 范围

- 新增 macOS App Bundle 打包脚本
- 新增 App Bundle 所需 `Info.plist`
- 新增打包校验脚本
- 补充 README 打包说明

## 约束

- 不改动现有运行逻辑
- 不主动处理 7777 端口占用
- 产物与前端 `dist/` 分离
- 首版不做代码签名和 notarization

## 验证

- 可执行打包脚本生成 `.app`
- App Bundle 包含 `Contents/MacOS`、`Contents/Info.plist`
- 主程序可通过 `file` 识别为 macOS 可执行文件
