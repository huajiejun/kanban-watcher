# 手机模式 Session 切换重连修复计划

**Goal:** 让浏览器手机模式下的内嵌 `kanban-watcher-card` 在工作区 `latest_session_id` 变化时，和桌面模式一样切换到新的会话 realtime 通道，避免弹窗卡在旧 session。

**Architecture:** 对齐桌面 `workspace-home` 的会话切换判定，在手机卡片的更新周期中比较当前选中工作区前后 session ID；当 session 发生变化时，重连会话 websocket，并按需刷新当前弹窗消息。

**Tech Stack:** TypeScript、Lit、Vitest
