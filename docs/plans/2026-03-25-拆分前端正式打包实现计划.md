# 前端正式打包拆分 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 Home Assistant 卡片构建与网页端正式构建拆分为独立命令和独立产物目录，避免误用同一个 `npm run build`

**Architecture:** 保留网页端开发服务器能力，新增 `vite.config.ha.ts` 与 `vite.config.web.ts` 分离两类产物；通过 `package.json` 暴露 `build:ha`、`build:web` 和聚合 `build` 命令；补充测试与文档保证行为清晰。

**Tech Stack:** Vite 7、TypeScript、Vitest、Lit
