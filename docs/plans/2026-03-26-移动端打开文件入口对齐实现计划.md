# 移动端打开文件入口对齐 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让移动端与桌面端打开文件时都进入对应工作区目录，而不是移动端退回 File Browser 根目录。

**Architecture:** 将工作区路径计算逻辑抽成共享方法，由桌面端工作区页与移动端卡片共同复用；先用单元测试锁住移动端未传 `workspacePath` 的回归，再补最小实现并验证两端一致。

**Tech Stack:** TypeScript, Lit, Vitest, jsdom
