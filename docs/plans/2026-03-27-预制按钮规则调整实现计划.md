# 预制按钮规则调整 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让本地 LLM 输出 AI AGENT 下一步可执行且容易的动作，并让禁止动作规则可配置，同时修复快捷按钮正则拆分不准的问题。

**Architecture:** 在快捷按钮核心逻辑中引入规则配置与统一归一化过滤流程。Decision prompt 动态拼装禁止动作约束；正则兜底从“识别标记”升级为“提取真实动作文本”。

**Tech Stack:** TypeScript, Lit, Vitest, Vite
