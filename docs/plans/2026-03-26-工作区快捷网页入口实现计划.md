# 工作区快捷网页入口 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为工作区表头新增快捷网页入口，点击后按端类型弹出 FRP iframe 页面，手机端全屏、宽屏端居中大面板。

**Architecture:** 在 `workspace-conversation-pane` 新增独立按钮和事件，在 `workspace-home` 新增页面级快捷网页弹层状态与渲染，并复用现有 `browser_url` 解析逻辑生成 iframe 地址。现有开发预览抽屉继续保留，新入口单独维护视觉层和关闭链路，避免语义混淆。

**Tech Stack:** Lit、TypeScript、Vitest、现有 `workspace-home`/`workspace-conversation-pane` 组件体系

---

## 文件结构

### 需要修改

- `src/components/workspace-conversation-pane.ts`
  - 新增快捷网页入口按钮属性、事件和表头按钮顺序。
- `src/web/workspace-home.ts`
  - 新增快捷网页弹层状态、打开/关闭逻辑、弹层渲染与边界清理。
- `src/styles.ts`
  - 新增快捷网页弹层样式，并区分手机端与宽屏端布局。
- `tests/workspace-home.test.ts`
  - 为新按钮、弹层布局和关闭边界补充行为测试。

### 可选新增

- 无。当前功能可以在既有文件边界内完成，不需要新增新的状态模块。

- [ ] **Step 3: 在 `src/web/workspace-home.ts` 写最小弹层渲染**

新增方法：

```ts
private handleCloseWebPreview = () => {
  this.webPreviewWorkspaceId = undefined;
};

private renderWebPreviewOverlay() {
  const workspace = this.workspaces.find((item) => item.id === this.webPreviewWorkspaceId);
  const previewUrl = workspace ? this.getWorkspaceEmbeddedPreviewUrl(workspace) : "";
  if (!workspace || !previewUrl) {
    return nothing;
  }

  const layout = this.mode === "mobile-card" ? "mobile" : "desktop";

  return html`
    <div
      class="workspace-home-web-preview-overlay"
      data-layout=${layout}
      @click=${this.handleWebPreviewOverlayClick}
    >
      <section class="workspace-home-web-preview-dialog" @click=${(event: Event) => event.stopPropagation()}>
        <header class="workspace-home-web-preview-header">
          <div class="workspace-home-web-preview-title">${workspace.name}</div>
          <button
            class="workspace-home-web-preview-close"
            type="button"
            @click=${this.handleCloseWebPreview}
          >
            ✕
          </button>
        </header>
        <iframe
          class="workspace-home-web-preview-frame"
          src=${previewUrl}
          title=${`${workspace.name} 快捷网页`}
        ></iframe>
      </section>
    </div>
  `;
}
```

并在 `render()` 末尾挂入 `${this.renderWebPreviewOverlay()}`。
