# 开发服务器子路径访问 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 支持通过 `https://dev.huajiejun.cn/<port>/` 访问当前 worktree 的前端开发服务器，并保证静态资源、HMR、API 与 realtime 请求都走同一子路径。

**Architecture:** 通过 `vite` 配置显式支持子路径 `base` 与 HMR path；运行时根据当前 URL 推导相对子路径 API base；`start-dev.sh` 在启动前端时注入当前端口对应的子路径环境变量。

**Tech Stack:** TypeScript, Vite, Vitest, Bash
