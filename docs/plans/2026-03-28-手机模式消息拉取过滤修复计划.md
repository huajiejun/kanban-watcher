# 手机模式消息拉取过滤修复计划

**Goal:** 让浏览器手机模式下内嵌 `kanban-watcher-card` 打开工作区弹窗时，与桌面模式一致，只拉取活跃对话需要的消息类型，避免落到陈旧历史消息。

**Architecture:** 对齐桌面 `workspace-home` 的 active pane 消息过滤规则，在手机卡片请求 `/api/workspaces/{id}/latest-messages` 时补充 `types` 参数；仅调整消息拉取请求，不改 websocket 协议与现有消息归并逻辑。

**Tech Stack:** TypeScript、Lit、Vitest
